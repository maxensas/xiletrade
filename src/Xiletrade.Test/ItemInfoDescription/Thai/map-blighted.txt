{ ม็อดแฝง }
ด่าน ติดเชื้อจาก ก้อนเชื้อร้าย (implicit)
ม็อดที่ส่งผลต่อ จำนวนของไอเทม จะส่งผลต่อจำนวน หีบไบล์ท เช่นกันในสัดส่วน 25% ของค่าดังกล่าว (implicit)
สามารถ เจิม ได้สูงสุด 3 ครั้ง — ไม่สามารถเปลี่ยนแปลงค่าตัวเลขได้ (implicit)
ผู้อาศัยตามธรรมชาติของด่านนี้ ถูกกำจัดออกไป — ไม่สามารถเปลี่ยนแปลงค่าตัวเลขได้ (implicit)
--------
เดินทาง​ไปยัง แผนที่นี้ ด้วย​การ​ใช้​ไอเทม​นี้​​ใน เครื่อง​เปิด​แผนที่ ของ​คุณ แผนที่​แต่​ละ​ชิ้น​สามารถ​ใช้ได้​เพียง​ครั้ง​เดียว​เท่านั้น
