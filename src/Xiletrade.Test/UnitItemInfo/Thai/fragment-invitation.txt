ชนิดไอเทม: ไอเทมแผนที่อื่นๆ
ความหายาก: ปกติ
Polaric Invitation
--------
เลเวลไอเทม: 83
--------
{ ม็อดแฝง }
ม็อดที่ส่งผลต่อ จำนวนของไอเทม ส่งผลต่อ จำนวนของ ของรางวัลที่บอสดรอป เช่นกัน (implicit)
--------
เมื่อ​อยู่​ใน​เงื้อมมือ​ของ​ดวงดาว​สีดำ
ย่อม​ไม่มี​ทาง​หลบ​หนี
--------
เปิด​วาร์ป​ไป​ยัง ขั้วของความว่างเปล่า ด้วย​การ​ใช้​ไอเทมชิ้น​​นี้​ใน เครื่อง​เปิด​แผนที่ ของ​คุณ
