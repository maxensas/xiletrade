Класс предмета: Микстуры
Редкость: Волшебный
Thin Borealwood Tincture of Absorption
--------
Накладывает Сжигание маны каждые 0,90 (augmented) секунд(-ы)
7 секунд(-ы) перезарядки при деактивации
--------
Требования:
Уровень: 32
--------
Уровень предмета: 84
--------
{ Собственное свойство — Урон, Стихийный, Холод, Атака, Состояние — -34% увеличение }
25% шанс заморозить оружием ближнего боя (implicit)
(Заморозка снижает скорость действий врагов до нуля, не давая им совершать действия. Длительность зависит от урона от холода нанесенного удара) (implicit)
27(25-35)% усиление эффекта охлаждения от оружия ближнего боя (implicit)
(Охлаждение снижает скорость действий максимум на 30%) (implicit)
--------
{ Префикс "Слабая"  (Уровень: 3)  — -34% увеличение }
34(35-31)% ослабление эффекта — Неизменяемое значение
50% снижение частоты активации Сжигания маны
{ Суффикс "присвоения"  (Уровень: 4)  — -34% увеличение }
Дарует 2(2-3) маны за каждого врага, убитого оружием ближнего боя
--------
Щёлкните ПКМ, чтобы активировать.  Одновременно может действовать только одна микстура. За каждый заряд Сжигания маны вы теряете 1% от максимального запаса маны в секунду. Может быть деактивирована вручную или автоматически деактивируется при достижении 0 маны.
